# 快捷键参考

OneMarkDotNet 通过全局键盘钩子（`KeyboardHook`）在 OneNote 窗口中拦截按键，实现 Markdown 渲染、导出和模式切换等快捷操作。本文档列出所有快捷键、Markdown 语法检测模式、Ribbon 按钮以及设置项对快捷键的影响。

## 全局快捷键

以下快捷键仅在 OneNote 窗口为前台窗口时生效。

### 渲染相关

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Enter` | 实时渲染 | 当前行包含 Markdown 语法时，自动触发渲染 |
| `F5` | 渲染当前页面 | 将当前页面内容解析为 Markdown 并渲染 |
| `Ctrl+Enter` | 退出源码模式 | 退出源码模式并立即渲染页面 |
| `Tab` | 表格/列表内渲染 | 在表格或列表中触发实时渲染 |

### 模式切换

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+,` | 切换源码/渲染模式 | 在源码模式和渲染模式之间切换 |

### 导入导出

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `F8` | 导出到剪贴板 | 将当前页面内容导出为 Markdown 并复制到剪贴板 |

## Markdown 语法检测

实时渲染仅在当前行匹配以下模式时触发。`MarkdownRenderHandler.ContainsMarkdownSyntax()` 方法逐行检查文本是否包含 Markdown 语法。

| 检测模式 | 正则表达式 | 匹配示例 |
|----------|-----------|----------|
| 标题 | `^#{1,6}\s` | `# 标题`、`## 二级标题` |
| 无序列表 | `^[-*+]\s` | `- 列表项`、`* 列表项`、`+ 列表项` |
| 有序列表 | `^\d+\.\s` | `1. 列表项`、`2. 列表项` |
| 引用 | `^>\s` | `> 引用内容` |
| 代码块 | `^```` | ` ```代码块 ` |
| 水平线 | `^(-{3,}\|\*{3,}\|_{3,})$` | `---`、`***`、`___` |
| 表格 | `\|.+\|` | `\| 列1 \| 列2 \|` |
| 加粗 | `\*\*.+\*\*` | `**加粗文本**` |
| 数学公式 | `\$\$.*\$\$` | `$$E=mc^2$$` |
| 任务列表 | `^-\s\[[ x]\]` | `- [x] 已完成`、`- [ ] 未完成` |
| 链接 | `\[.+\]\(.+\)` | `[链接文本](URL)` |
| 图片 | `!\[.*\]\(.*\)` | `![替代文本](图片URL)` |
| 行内代码 | `` `.+` `` | `` `代码` `` |

> **注意**：实时渲染需要 `IsRealtimeRenderEnabled` 设置为 `true`，且当前不在源码模式中。

## Ribbon 按钮

OneMarkDotNet 在 OneNote 功能区添加了 **OneMarkDotNet** 选项卡，包含以下按钮：

### Markdown 组

| 按钮 | 功能 | 对应快捷键 |
|------|------|-----------|
| Render | 渲染当前页面的 Markdown 内容 | `F5` |
| Export | 将 Markdown 导出到剪贴板 | `F8` |
| Source Mode | 切换源码/渲染模式 | `Ctrl+,` |

### Theme 组

| 按钮 | 功能 | 对应快捷键 |
|------|------|-----------|
| Theme（动态菜单） | 选择主题 | — |
| Open Directory | 打开用户主题目录（`%APPDATA%\OneMarkDotNet\themes\`） | — |
| Reload | 重新加载所有主题文件 | — |

### File 组

| 按钮 | 功能 | 对应快捷键 |
|------|------|-----------|
| Import | 从文件导入 Markdown | — |
| Export to File | 导出 Markdown 到文件 | — |

### Help 组

| 按钮 | 功能 | 对应快捷键 |
|------|------|-----------|
| About | 显示关于对话框（版本信息） | — |

## 设置影响

以下设置项存储在 `%APPDATA%\OneMarkDotNet\settings.json` 中，会影响快捷键和渲染行为。

| 设置项 | 类型 | 默认值 | 影响的快捷键 | 说明 |
|--------|------|--------|-------------|------|
| `IsRealtimeRenderEnabled` | `bool` | `true` | `Enter`、`Tab` | 关闭后 `Enter` 和 `Tab` 不再触发实时渲染 |
| `IsLineNumberEnabled` | `bool` | `true` | — | 控制代码块是否显示行号（与 `--enable-code-line-number` 主题变量配合使用） |
| `IsLatexToImage` | `bool` | `true` | — | 控制数学公式是否渲染为图片（与 `--enable-latex-to-image` 主题变量配合使用） |
| `IsSourceModeDefault` | `bool` | `false` | — | 控制新块是否默认为源码模式 |
| `CurrentThemeName` | `string` | `"dark"` | — | 当前使用的主题名称，切换主题后自动保存 |

## 自定义快捷键

当前版本的快捷键硬编码在 `KeyboardHook.cs` 中，不支持通过配置文件修改。

### 修改方式

1. 编辑 `src\OneMarkDotNet\KeyboardHook.cs` 中的 `HookCallbackProcedure` 方法
2. 修改按键判断逻辑，例如：
   - 修改 `VK_F5`（`0x74`）为其他虚拟键码
   - 修改修饰键组合（`ctrlPressed`、`shiftPressed`、`altPressed`）
   - 添加新的按键事件
3. 重新构建项目并注册 Add-in 后生效

### 虚拟键码参考

`KeyboardHook.cs` 中已定义的虚拟键码常量：

| 常量名 | 值 | 对应按键 |
|--------|-----|---------|
| `VK_F5` | `0x74` | F5 |
| `VK_F8` | `0x77` | F8 |
| `VK_RETURN` | `0x0D` | Enter |
| `VK_TAB` | `0x09` | Tab |
| `VK_OEM_COMMA` | `0xBC` | `,`（逗号） |

### 添加新快捷键的步骤

1. 在 `KeyboardHook.cs` 中定义新的虚拟键码常量
2. 在 `HookCallbackProcedure` 方法中添加按键判断逻辑
3. 定义新的事件（如 `NewShortcutRequested`）
4. 在 `OneMarkAddIn.cs` 中订阅该事件并实现处理逻辑
5. 重新构建并注册
