# 快捷键参考

## 全局快捷键

NoteMark 通过全局键盘钩子拦截按键，以下快捷键在 OneNote 窗口激活时生效。

### 渲染相关

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Enter` | 实时渲染当前行 | 需在设置中启用实时渲染，且当前行内容需匹配 Markdown 语法模式 |
| `F5` | 渲染选中区域或当前行 | 如果有选中文本则渲染选区，否则渲染当前行 |
| `Ctrl+Enter` | 退出源码模式并渲染 | 将当前块从源码模式切换为渲染模式并立即渲染 |
| `Tab` | 表格/列表内渲染 | 仅在当前行处于表格或列表上下文中时触发渲染 |

### 模式切换

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+,` | 切换源码/渲染模式 | 如果当前块在渲染模式则切换为源码模式，反之亦然 |

### 导入导出

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `F8` | 导出到剪贴板 | 将当前 OneNote 页面内容转换为 Markdown 并复制到剪贴板 |

## Markdown 语法检测

实时渲染仅在当前行匹配以下模式时触发：

| 模式 | 示例 |
|------|------|
| 标题 | `# Heading`、`## Heading` |
| 无序列表 | `- item`、`* item`、`+ item` |
| 有序列表 | `1. item` |
| 引用 | `> quote` |
| 代码块 | ` ``` ` |
| 水平线 | `---`、`***`、`___` |
| 表格 | `\| col1 \| col2 \|` |
| 加粗 | `**bold**`、`__bold__` |
| 数学公式 | `$$formula$$` |
| 任务列表 | `- [ ] task`、`- [x] done` |
| 链接 | `[text](url)` |
| 图片 | `![alt](url)` |
| 行内代码 | `` `code` `` |

## Ribbon 按钮

除快捷键外，以下功能可通过 OneNote 功能区的 NoteMark 选项卡访问：

| 按钮 | 功能 | 对应快捷键 |
|------|------|-----------|
| Render Markdown | 渲染当前选中区域或行 | F5 |
| Export Markdown | 导出当前页面到剪贴板 | F8 |
| Source Mode | 切换源码/渲染模式 | Ctrl+, |
| Theme → Select Theme | 选择主题 | — |
| Theme → Open Theme Directory | 打开主题文件目录 | — |
| Theme → Reload Themes | 重新加载主题 | — |
| Import Markdown | 从 .md 文件导入 | — |
| Export to File | 导出到 .md 文件 | — |
| About | 关于对话框 | — |

## 设置影响

| 设置项 | 影响的快捷键 | 说明 |
|--------|-------------|------|
| `isRealtimeRenderEnabled` | Enter、Tab | 关闭后 Enter 和 Tab 不再触发实时渲染 |
| `isLineNumberEnabled` | — | 控制代码块是否显示行号 |
| `isLatexToImage` | — | 控制数学公式是否渲染为图片 |
| `isSourceModeDefault` | — | 控制新块是否默认为源码模式 |
